# Monitor-Window

针对AI或其他用户来说，需要等待AI生成完成的时间是不确定的，如果一直守着将会浪费自己金贵的时间，此时需要有个小工具在任务执行完成后进行提醒。


该小工具可以对窗口进行截图，比对方式有两种：
- 关键字：各词之间以空格分割，当截图识别为文本后包含所有词时认为认为完成，此时会给出声音提示
- 图像对比：对于某些不方便识别文字或结果不确定时，可以通过图像对比的方式进行识别，若N秒后图像无变化则认为任务已完成，此时会给出声音提示
